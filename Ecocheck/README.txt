# 🌱 EcoCheck - Conscious Consumer Platform

EcoCheck is a web application that helps consumers make informed, sustainable purchasing decisions by providing instant environmental impact ratings for brands.

## 📋 Overview

EcoCheck empowers conscious consumers to quickly verify whether a brand aligns with their sustainability values. Simply search for any brand to get instant feedback on its environmental practices, sustainability rating, and eco-certifications.

## ✨ Features

- **Instant Brand Search**: Quick lookup of brands with real-time sustainability ratings
- **Sustainability Ratings**: 1-5 star rating system for environmental impact
- **Eco-Certifications Display**: Shows relevant environmental certifications
- **Sustainable Brands Showcase**: Curated list of eco-friendly brands
- **Mobile Optimized**: Fully responsive with touch-friendly interactions
- **Performance Optimized**: Smooth animations and fast loading on all devices
- **URL Search Parameters**: Share brand searches via URL links

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Node.js with Express
- **API**: RESTful endpoints for brand data
- **Styling**: Bootstrap 5 + Custom CSS
- **Icons**: Font Awesome 6
- **Performance**: Intersection Observer API, Debounced search
